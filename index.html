i write python scripts to randomise playback of the input signal like a granular looper
the scripts give me some control over playback speed, direction, volume, etc., but always with uncertainty
all my code is amateur, serves only limited purpose and very specific hardware, being controlled by computer keyboards with stickers all over them, so is not immediately useful to anyone
but i am happy to talk about how things work and show you the code if you like, maybe one day i will put it up on github
i started by wanting to build a looper that could handle several loops of unrelated lengths, and it snowballed
pyo does the music
blessings' child blessed tells me what's going on in the terminal-based gui
python-rtmidi lets me control it with a midi keyboard
evdev let me control it with multiple computer keyboards
